🔧 CHECK STATUS & TROUBLESHOOT
═════════════════════════════════════════════════════════════════════════════════

Frontend not starting?
  → Check: Is Node.js installed? (node --version)
  → Check: Are you in right folder? (cd "D:\Forensics Application")
  → Try: Delete node_modules/ and run npm install again

Can't access http://localhost:5173?
  → Check the PowerShell window for errors
  → Make sure npm run dev finished without errors
  → Try a different port: npm run dev -- --port 5174

API calls not working?
  → Make sure Flask backend is running
  → Check http://localhost:5000 in browser

Need help?
  → Read: REACT_FRONTEND_SETUP.txt (troubleshooting section)
  → Read: FLASK_REACT_INTEGRATION.txt (backend issues)
